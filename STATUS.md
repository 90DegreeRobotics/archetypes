- No ambient/music layer, Chronos lineage/mutation or world memory, or full Windows installer exists yet. Persistence currently covers the Witness profile, last artifact receipt, generated audio cache, and the local Lane 0 ledger; it is not yet a full world-memory system.
- **Per-line TTS latency (found 2026-07-12 evening):** each council line's speech spawns a fresh `sherpa-onnx-offline-tts.exe` process and reloads the Kokoro model rather than reusing a warm/persistent server; a line can take longer than ~4-5 seconds to actually start playing. The UI now honestly shows "forming speech…" for that whole gap (see the sync fix above) instead of masking it, but the latency itself is unaddressed.
- **Table geometry:** the physical table shell (top, rims, legs — not the `Stargate_Portal` disc, which is confirmed good) still reads as thin and hard to see against the chamber's near-black lighting. Blocked on the operator's reference image.
