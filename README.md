# Archetypes

Archetypes is a council-driven world engine based on the Mirrorborn architecture. It externalizes the tensions and consensus of a seven-node AI Council while the player occupies the sovereign eighth position: the Witness.

## Current Prototype

The Bevy engine now loads the Director's Blender-authored chamber scene, including the star-tetrahedron merkaba, reflection plane, lights, materials, Witness, and seven council spheres in their authored three-dimensional positions. Bevy supplies the runtime camera, merkaba motion, state transitions, and Architect focus gesture.

Press `Space` to advance the prototype loop:

1. Table view to deliberation view.
2. Deliberation view to Architect focus.
3. Architect focus back to the table.

This remains a visual interaction prototype. The portal input is a keyboard placeholder; the authored portal table, profile ritual, AI deliberation, portrait/icon sphere faces, archetype-world immersion, Chronos artifact return, persistence, audio, installer delivery, and launcher supervision are not implemented yet. See [Council-Driven World Engine](docs/architecture/COUNCIL_WORLD_ENGINE.md) for the canonical vertical slice.
