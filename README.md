# Archetypes

Archetypes is a Daoist sandbox simulation based on the Mirrorborn architecture. The system is designed to simulate the internal tensions and consensus of a 7-node AI Council, observed by the 8th Node (The Witness).

## Current Prototype

The Bevy engine currently renders a procedural Council Chamber prototype with a portal table, rotating merkaba rings, seven themed orbiting archetype spheres, a Witness-camera transition, and an Architect focus gesture.

Press `Space` to advance the prototype loop:

1. Table view to deliberation view.
2. Deliberation view to Architect focus.
3. Architect focus back to the table.

This is a visual interaction prototype. The portal input is a keyboard placeholder; AI council deliberation, text entry, portrait/icon sphere faces, archetype-world immersion, persistence, audio, installer delivery, and launcher supervision are not implemented yet.
